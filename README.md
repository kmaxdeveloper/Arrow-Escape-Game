Designed for speed, flexibility, and zero setup pain, this package gives you everything needed to prototype or ship a polished puzzle game without building core systems from scratch.

## 🚀 Features

### 🧩 Core Gameplay
- Fully functional grid based system  
- Smart arrow movement and collision detection  
- Smooth sliding mechanics  
- Automatic exit logic when path is clear  

---

### 🎨 Customization and Tools
- 7 plus ready to use themes  
- Pre made levels included  
- Built in level editor using ScriptableObject  
- Theme editor for fast visual changes  
- Mobile ready controls out of the box  

---

### 💰 Monetization Ready
- AdMob ads integration included  

---

## ⚡ Why Arrow Escape

This template removes the boring setup work and lets you focus on what actually matters

✔ Designing fun levels  
✔ Experimenting with mechanics  
✔ Shipping faster  

No need to build grid systems, movement logic, or editors from scratch  

---

## 🧠 How It Works

- Define your grid size  
- Set move limits  
- Place arrows using the editor  
- Hit play and your level is ready  

No coding required
